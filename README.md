# 📊 PRODIGY_DS_04 – Sentiment Analysis on Social Media Data

## 📌 Overview

This project was completed as part of my **Data Science Internship at Prodigy InfoTech**.

The objective is to analyze sentiment patterns in social media text to understand public opinion and classify text into different sentiment categories.

---

## 🎯 Objective

Build a sentiment analysis pipeline that:

- Cleans textual data
- Performs Exploratory Data Analysis (EDA)
- Visualizes sentiment distribution
- Generates Word Clouds
- Extracts frequently used words
- Evaluates sentiment patterns

---

## 📂 Dataset

Twitter Sentiment Dataset

Columns:

- ID
- Topic
- Sentiment
- Tweet
